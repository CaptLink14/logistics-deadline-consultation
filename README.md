# logistics-deadline-consultation
Aplicação web desenvolvida com Python e Streamlit para resolver um problema real da equipe de Transporte e Planejamento: a falta de visibilidade centralizada sobre os prazos das transportadoras parceiras.

Web application developed with Python and Streamlit to solve a real problem faced by the Transportation and Planning team: the lack of centralized visibility over partner carriers’ delivery timelines.

### Problema | Problem
A equipe confirmava prazos manualmente transportadora por transportadora, gerando retrabalho e inconsistências.

The team manually checked deadlines carrier by carrier, leading to rework and inconsistencies.

### Solução | Solution
Dashboard interativo com filtros por transportadora, cidade e UF, consolidando todas as tabelas em uma interface única.

An interactive dashboard with filters by carrier, city, and state, consolidating all tables into a single interface.

### Tecnologias | Technologies
Python | Pandas | Streamlit | Excel/OpenPyXL

### Resultado | Results
10 pessoas da equipe passaram a usar a ferramenta 
no dia a dia, eliminando consultas manuais recorrentes.

10 people from the team started using the tool in their daily routine, eliminating recurring manual queries.
